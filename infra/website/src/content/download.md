## System Requirements

### Minimum Requirements
- 4 GB RAM (8 GB recommended for large files)
- 500 MB available disk space (plus space for temporary files during sorting)
- Modern multi-core processor

### Recommended for Best Performance
- 16 GB RAM or more for handling 5-10 GB data files
- SSD for faster temporary file operations
- Multi-core processor (4+ cores)

## Free vs Premium

BreachLine is available in two versions:

### Free Version
- Load files up to 1 GB
- Basic filtering and search
- Timeline visualization
- Cross-platform support
- Perfect for smaller investigations

### Premium Version
- Handle files up to 5-10 GB
- Advanced SPL-like query language
- Workspace annotations and export
- Enhanced performance features
- Priority support
- Lifetime license
